# CodeAlpha_Decision-Prediction-from-Medical-Data
# Disease Prediction from Medical Data

## Overview
This project predicts the possibility of a disease using machine learning classification algorithms on structured medical data. It uses the Breast Cancer Wisconsin dataset from Scikit-learn and compares the performance of multiple classifiers.

## Features
- Predicts disease based on medical features.
- Uses structured medical dataset.
- Compares multiple machine learning algorithms.
- Evaluates model performance using standard metrics.
- Beginner-friendly and easy to understand.

## Technologies Used
- Python
- Scikit-learn
- NumPy
- Pandas

## Machine Learning Algorithms
- Logistic Regression
- Support Vector Machine (SVM)
- Random Forest Classifier

## Dataset
**Breast Cancer Wisconsin Dataset**

- Source: Scikit-learn Dataset
- Number of Samples: 569
- Number of Features: 30
- Classes:
  - Malignant
  - Benign

## Evaluation Metrics
- Accuracy Score
- Classification Report
- Confusion Matrix

## Installation

Clone the repository:

```bash
git clone https://github.com/saisribhavitha-arch/CodeAlpha_Disease-Prediction-from-Medical-Data.git
```

Install dependencies:

```bash
pip install scikit-learn numpy pandas
```

## Run the Project

```bash
python disease_prediction.py
```
